# ttgenie

## Timetable Genie

Converts data from the XLSX format into HTML for quick review.

## Build

1. Update the version in `pubspec.yaml`
2. Generate the Web App: `flutter build web --base-href /ttgenie/`

## Use

Access the Web App at: [https://npaspallis.github.io/ttgenie/](https://npaspallis.github.io/ttgenie/)
